# Pview

Pview is a interactive paper viewer for the researchers with LLM powered. It provides a simple and efficient way to view and manage research papers. It supports PDF files and allows users to organize their papers in a hierarchical structure. Users can also add notes and tags to their papers for easy retrieval.

## Features

- View PDF files with a built-in PDF viewer
- Organize papers in a hierarchical structure
- Add notes and tags to papers
- Search papers by title, author, or tags
- Chat with an LLM to get insights and summaries of papers
